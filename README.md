## Installation

### Windows (EXE)
1. Download `SD_Card_Tester_Pro.exe`
2. Run as an Administrator

### Linux (DEB)
```bash
# Install the package
sudo dpkg -i sd-card-tester-pro.deb

# Install dependencies (if necessary)
sudo apt-get install -f

# Launch the program
sd-card-tester-pro
```
### Linux (Flatpak)
```bash
# Install Flatpak
flatpak install flathub com.sdcardtester.pro
```

### MacOS (.app bundle):
1. Download SD_Card_Tester_Pro.app

2. Drag and drop to the Applications folder

3. Launch from the Finder (Security & Privacy permission may be required)
